How to Use This Book

This book teaches the Kotlin language to both programming beginners and those who have already programmed in another language.

Beginners: Start with the Introduction and move through each chapter (we call chapters atoms because they’re so small) as you would any other book—including the Summary 1 and Summary 2 atoms, which solidify your knowledge.

Experienced Programmers: Because you already understand the fundamentals of programming, we have prepared a “fast track”:

  1. Read the Introduction and Examples & Exercises.

  2. Jump forward to Summary 1; read it and solve the exercises.

  3. Jump forward to Summary 2; read it and solve the exercises.

  4. At this point, continue normally through the book, starting with Section III: Usability.
