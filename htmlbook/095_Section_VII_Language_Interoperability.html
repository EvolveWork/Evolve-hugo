Section VII: Language Interoperability

This section explores the issues and techniques involved when interfacing between Kotlin and Java.

If you’re not a Java programmer, feel free to skip this section.

An essential Kotlin design goal is to create a seamless programming experience for Java programmers. If you want to slowly migrate to Kotlin, you can easily start by sprinkling bits of Kotlin among your existing Java project. This way you can write new Kotlin code atop an existing Java base, benefiting from Kotlin language features without being forced to rewrite your Java code.
